** English

This document explains how to install and run Too Many Ways app on your Android device.

Once downloaded the zip package and extracted it, run toomanyways.apk file to install. Once this is done, run it from your apps menu on your device.

If you're going to check or mess with the code, you just need to import the project (once extracted) on Android Studio. (File...>New...>Import Project...>PathToProjectFolder).

Usage:

Insert your experience in IT (in years), to display a suitable menu for you with the best resources to learn 5 different study fields in computer science.

** Español

Este documento explica como instalar y correr la app Too Many Ways en tu dispositivo Android.

Una vez descargado el paquete zip, ejecuta el archivo toomanyways.apk para proceder a la instalación. Una vez hecho, podrás correr la app desde el menú apps de tu dispositivo.

Utilización:

Inserta tu experiencia en TI (en años) para proporcionarte un menú adecuado con los mejores recursos para aprender 5 campos de estudio diferentes dentro de las ciencias de la computación.

Si quieres estudiar el código y cacharrear con el, tan sólo necesitas importar el proyecto (una vez extraido) a Android Studio. (Archivo...>Nuevo...>Importar Proyecto...>RutaACarpetaDeProyecto)
